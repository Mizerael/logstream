# logstream-course

Для запуска решений:
```bash
rye sync
source .venv/bin/activate
# Для запуска решений 
rye run комманда_решения
# Для отображения списка всех комманд
rye run -l 
# или
rye run python3 src/blockN/файл
```
Для установки `rye` можно перейти по [ссылке](https://rye.astral.sh/guide/installation/).
